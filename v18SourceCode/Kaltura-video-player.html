HTML source for the four examples above.


On a standalone link trigger

<p class="ibm-standalone-link"><a data-widget="videoplayer" data-videodisplay="overlay" data-videoid="0_orew0puw" data-videotype="kaltura" data-showvideoduration="true" class="ibm-video-link" href="####SOMEURL">Watch the video</a></p>

On an inline link

<p>Hi, I think you should really <a data-widget="videoplayer" data-videodisplay="overlay" data-videoid="0_orew0puw" data-videotype="kaltura" href="####SOMEURL">watch this video</a> because it's awesome.</p>

On a (custom) image link (with optional theater mode)

<p><a data-widget="videoplayer" data-videodisplay="overlay" data-videoid="0_orew0puw" data-videotype="kaltura" data-theatermode="true" href="####SOMEURL"><img class="ibm-resize" src="sample-image.png" width="220" height="120" alt="Just a placeholder image"></a></p>

Using an auto-gen'd image

<p data-widget="videoplayer" data-videodisplay="overlayandtrigger" data-videoid="0_orew0puw" data-videotype="kaltura" href="####SOMEURL">Default content here.</p>
